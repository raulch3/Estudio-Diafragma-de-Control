# Estudio-Diafragma-de-Control
Aplicando SQL y Excel

La importancia del estudio del control estadístico de procesos radica en que permite observar y entender cómo se comporta un proceso a lo largo del tiempo. A través del uso de límites superiores e inferiores nos identificar variaciones en indicadores como tasas, producción o promedios de ventas.
